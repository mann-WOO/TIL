# PEP8

> PEP8 : 스타일가이드



###  string quote

small or big quotation mark

- 작은 따옴표와 큰 따옴표 중 하나로만 통일하여 사용한다.
- 작은 따옴표로 쓰는 것이 더 보편적이다.



### indentation

4 spaces or tab

- 4 spaces와 tab 중 하나로 통일한다
- 4 spaces를 쓰는 것이 더 보편적이다.



### naming

variable/ function

- snake_case

class

- PascalCase



### space

function(para1='A', para2='B')

- 함수 등의 괄호 안에서 연산자 앞뒤로는 붙여쓴다

a = 3 + 5 

- 변수 할당 시 연산자 앞뒤로는 띄어 쓴다.
